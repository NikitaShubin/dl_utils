# Пример таблицы меток [`labels_template.xlsx`](../labels_template.xlsx "Перейти к файлу")

## Назначение

Файл `labels_template.xlsx` используется для определения иерархической структуры
меток объектов с их названиями и синонимами.

## Структура файла

### Основные столбцы

- **Класс объекта** (столбец A): Иерархическая структура меток с нумерацией
- **Метка в CVAT** (столбец C): Основное название метки для использования в CVAT
- **Метка в другом источнике данных** (столбец E): Альтернативные названия/синонимы

### Правила заполнения

#### Нумерация

- **Римские цифры** (I, II, III): Используются для группировки меток
- **Арабские цифры** (1, 1.1, 1.1.1): Определяют иерархию внутри групп
- **Обязательная точка** в конце номера: `1.`, `1.1.`, etc.

#### Пример структуры

```text
I. Домен                 -> domain
1. Царство              -> kingdom  
1.1. Отдел              -> phylum
1.1.1. Класс            -> class
1.1.1.1. Порядк (отряд) -> order
```

### Особенности

- Метки в CVAT должны быть уникальными
- Синонимы позволяют использовать альтернативные названия меток
- Поддерживается многоуровневая иерархия меток
- Автоматическое построение дерева меток при загрузке
