# Пример таблицы суперметок [`superlabels_template.xlsx`](https://github.com/NikitaShubin/dl_utils/blob/main/superlabels_template.xlsx "Перейти к файлу")

## Назначение
Файл `superlabels_template.xlsx` определяет суперметки - группы меток, объединенные по определенному признаку.

## Структура файла

### Основные столбцы:
- **№ п/п** (столбец A): Номер суперметки
- **Наименование суперметки** (столбец B): Название группы меток
- **Метки** (столбец C): Список меток, входящих в суперметку
- **Приоритет** (столбец D): Приоритет суперметки

### Преобразование номеров:
Исходный номер → Фактический номер суперметки:
- `1` → `0`
- `2` → `1`
- `0` → `-1` (неиспользуемые объекты)
- `-1` → `-2` (исключаемые объекты)

### Специальные суперметки:

#### Неиспользуемые объекты (номер -1):
- Объекты остаются в разметке, но игнорируются при создании датасета
- Пример: "Отдел" в биологической классификации

#### Исключаемые объекты (номер -2):
- Кадры, содержащие такие объекты, полностью исключаются из датасета
- Пример: "Домен" в биологической классификации

### Пример группировки:
```
Суперметка "Виды и рода" (номер 0) включает:
- Род
- Вид

Суперметка "Классы, отряды и семейства" (номер 1) включает:
- Класс
- Порядок (Отряд) 
- Семейство
```

### Особенности:
- Одна метка может входить только в одну суперметку
- Номера суперметок используются в YOLO-формате
