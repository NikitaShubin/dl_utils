# Модуль [`sam3al.py`](../sam3al.py)

Предоставляет инструменты для работы с Segment Anything Model 3 (SAM3) и интеграции
с системой интерактивной сегментации. Основные функции - обработка видео,
управление подсказками для сегментации и экспорт результатов в CVAT.

На текущем этапе модуль только начинает наполняться функциональностью, так что
единственная реализованная в нём функция сейчас — автоматическое получение весов
модели:

---

## Основные возможности

## Автономный запуск

Модуль можно запустить как скрипт для предварительной загрузки модели в папку по
умолчанию:

```bash
python sam3al.py
```

После выполнения в `~/models/` появится файл `sam3.pt`.

---

## Примечания

- Для работы загрузки требуется установленный пакет `modelscope`.  
  Установить можно командой:

  ```bash
  pip install modelscope
  ```

- Модель занимает около 3.21 ГБ (файл `.pt`).
- При повторном вызове с тем же путём загрузка не производится, если файл уже существует.
